    Ajax Class

    Ajax模块，特点：与$.ajax()使用方式类似，并自带功能。
    自带功能如下：
    1检测合法性，包括姓名，手机号码，邮箱地址，年龄，身份证号等
    2 避免同一地址在数据未返回之前重复提交
    3 自带提交中，请勿重复提交等提示。可设置是否可见

    <!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>HDjs.Ajax模块，和ajax用法一样，功能更强大</title>
    </head>
    <body>
        <p></p>
    </body>
    <script>
        //和平时写ajax格式是一样的喔
        HDjs.Ajax({
            url : './index.php?g=sanjiuchunwan&m=User&a=index',
            data:{
                name: "我是婷婷",
                age: 18
            },
            success:function(obj){
                console.log("提交成功，数据为："+obj);
                $("p").append('提交成功');  
            },
            error:function(obj){
                console.log("提交失败，数据为："+obj);
                $("p").append("提交失败，数据为："+obj);   
            }
        },false);


        //也可以这样写写，和以上写法一样
        var mydata = new Object();
        mydata.url = './index.php?g=sanjiuchunwan&m=User&a=index';
        mydata.data = {
            name: "婷婷",
            age: 18
        };
        mydata.succeed = function() {
            console.log("提交成功，数据为："+obj);
            $("p").append('提交成功');  
        };
        mydata.error=function(obj){
            console.log("提交失败，数据为："+obj);
            $("p").append("提交失败，数据为："+obj);   
        };


        HDjs.Ajax(mydata);  //测测重复提交


    </script>
</html>

    数据提交，只需要设置相应属性便可提交。自带检测姓名手机号码

    Parameters:

    nametypeflagdescription
    myData Object

    用于配置 Ajax 请求的键值对集合。data参数中，对name进行中文名验证，对phone进行手机号码验证，对age进行年龄验证，对email进行邮箱地址验证，对idCard进行身份证号验证。

    tips Boolean

    默认值：true。是否出现“数据提交中”，"请勿重复提交数据"等半透明底弹出框提示语。
