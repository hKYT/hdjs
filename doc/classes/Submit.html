    Submit Class

    提交模块，特点：无需记住ajax属性，并自带功能。HDjs.Ajax为基础。
    自带功能如下：
    1检测合法性，包括姓名，手机号码，邮箱地址，年龄，身份证号等
    2 避免同一地址在数据未返回之前重复提交
    3 自带提交中，请勿重复提交等提示

    Example:

    • Submit
    <!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>提交模块，不用配置ajax属性</title>
    </head>

    <body>
        <p></p>
    </body>
    <script>
        var url = './index.php?g=sanjiuchunwan&m=User&a=index'
        var succcess = function(obj) {
            console.log("提交成功，数据为：" + obj);
            $("p").append('提交成功');
        }
        //(myUrl,mySuccess,mydata,mytype,tips,other)

        HDjs.Submit(url, succcess, {
                name: "婷婷",
                age: 18
            },
            "GET",
            true, {
                error: function(obj) {
                    console.log("提交失败，数据为：" + obj);
                    $("p").append("提交失败，数据为：" + obj);
                }
            })

        //再提交一次
        HDjs.Submit(url, succcess);

    </script>
</html>

    数据提交，只需要设置相应属性便可提交。自带检测姓名手机号码

    Parameters:

    nametypeflagdescription
    myUrl String

    发送请求的地址

    mySuccess Function

    请求成功后的回调函数。

    mydata Object

    发送到服务器的数据。对name进行中文名验证，对phone进行手机号码 验证，对age进行年龄验证，对email进行邮箱地址验证，对idCard进行身份证号验证。

    mytype String

    默认值: "GET"。请求方式 ("POST" 或 "GET")。

    tips Boolean

    默认值：true。是否出现“数据提交中”，"请勿重复提交数据"等半透明底弹出框提示语。

    other Object

    设置ajax其他参数，该属性优先级最高。
