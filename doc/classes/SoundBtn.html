    音乐按钮

    Parameters:

    nametypeflagdescription
    opt Object

    图片加载的相关数据设置
    id 背景音乐模块的id，可以对此id做相应的样式，无需在页面中写入此id；此音乐模块会插入到body中。
    src 背景音乐的链接。
    onImg 音乐播放时的播放按钮图片链接，不设置时使用默认按钮图片。
    offImg 音乐暂停时的暂停按钮图片链接，不设置时使用默认按钮图片。
    styleCss 是否使用默认的播放按钮图片的动画效果，类型：boolean;默认为true，动画效果为顺时针旋转。
    className 自定义播放时的图片的动画的类名，仅在styleCss=false时生效。
    autoPlay 背景音乐是否自动播放，类型：boolean;默认为true，即自动播放。
