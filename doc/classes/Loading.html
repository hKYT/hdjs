    图片加载

    Parameters:

    nametypeflagdescription
    opt Object

    图片加载的相关数据设置
    imgUrl 图片链接数组
    defau 是否使用默认加载样式，默认为true
    loadColor 加载页面的背景颜色，默认为“#15191f”
    srcUrl 加载页面中的加载动画gif/图片链接，不设置时使用默认gif
    width 自定义加载动画gif/图片的宽度，不设置时默认为“800px”
    color 加载进度字体颜色，默认为“#fff”
    loadComplete 加载完成后的操作
    setLoadingInfo 加载时的操作，参数progress为加载进度
